Create new show allows for the creation of a new show that can be used in the audition and casting process. All fields are required.

  • Show Information: Set the concert type for the show, the quarter the show will occur in, and the end date and time.
  • Audtion Information: Set the audition name, location, and date/time of the show's audition.
