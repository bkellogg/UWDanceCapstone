Create New User Role

Current Roles:

Create new user role allows an admin to create a new roles with unique names and a permission level of 10, 70, or 100. All fields are required.

  • Role Name: Abbreviated or shorthand name for role.
  • Display Name of Role: Display name of role that will be viewed throughout the STAGE application.
  • Permission Level: This value assigns the user specific permissions and functionality within the STAGE application.
    • 100: Allows the user to have access to all the functionality of the STAGE application, view all user information, and access to the Admin Console.
    • 70: Allows the user to view user profiles, participate in the casting process, and modify all information about thier piece.
    • 10: Allows the user to audition for a show and modify their profile.
